Install of library for Paralax Feedback 360 Servo is needed and can be found here:
https://github.com/HyodaKazuaki/Parallax-FeedBack-360-Servo-Control-Library-4-Arduino
1. Download .ZIP library from github
2. launch Arduino IDE
3. Open Rotator Code File
4. Click Sketch > Include Library > Add .zip Library
5. Select .zip file in the pop up file explorer window
6. Write Arduino Code to board
